# Programming for Data Analytics -> Practical Assignments
by Stephen Kerr

## Purpose of this Repository
This repository contains the practical assignments for the module ***Programming for Data Analytics*** taught by Andrew Beatty as part of the HDIP in Computing and Data Analytics. To learn more information on the HDIP review the link here: [HDIP in Computing and Data Analytics](https://www.atu.ie/courses/higher-diploma-in-science-data-analytics)

Throughout the semester there were a mix of small practical assignments /  mini projects. Each assignment folder includes the relevant notebook or script, along with the raw data, processed outputs and generated figures or processed data.

## Technologies Used
- Python 3.12.1
- Git & Github
- Visual Studio Code

## Assignments Overview:

### Assignment 02 - Bank Holidays
- A Python script that loads and processes UK bank holiday data (.json), performing simple parsing and date handling to extract bank holidays unique to Northern Ireland. For a direct link to this Assignment click [here](https://github.com/skerr17/programming-for-data-analytics/tree/main/assignments/assignment02)

### Assignment 03 - Email Domain Analysis
- A notebook that extracts email domains from a dataset, computes frequency counts, and visualises the distribution. For a direct link to this Assignment click [here](https://github.com/skerr17/programming-for-data-analytics/tree/main/assignments/assignment03)

### Assignment 05 - Population Analysis
- A notebook that performs data cleaning and demographic analysis on Irish CSO population data, including age/sex breakdown, weighted means, and comparative plots. For a direct link to this Assignment click [here](https://github.com/skerr17/programming-for-data-analytics/tree/main/assignments/assignment05)

### Assignment 06 - Knock Airport Weather
- A notebook that performs analysis of hourly weather data including temperature trends, daily/monthly averages, wind speed cleaning, rolling averages and maximum wind speed visualisation. For a direct link to this Assignment click [here](https://github.com/skerr17/programming-for-data-analytics/tree/main/assignments/assignment06)
